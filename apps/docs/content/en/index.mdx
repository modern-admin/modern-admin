# Modern Admin

Universal admin panel framework, AdminJS-inspired but rebuilt on a modern
TypeScript stack:

- **NestJS 11** for the backend (REST + GraphQL + WebSocket).
- **React 19 + Vite 8 + TanStack Router + shadcn/ui + Tailwind 4** for the
  frontend.
- **Prisma 7** and **Drizzle 0.45** as plug-in ORM adapters.
- **Better Auth 1.6+** for authentication.
- **Redis** for cross-instance cache and realtime fan-out.
- **Zod 4** for end-to-end runtime validation.
