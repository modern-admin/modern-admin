Add `@modern-admin/skills` — Claude Code skill package distributed via
[`npm-skills`](https://github.com/bluelibs/npm-skills). Ships the
`modern-admin-integration` skill (one `SKILL.md` + eight topical
references) mirroring `apps/docs/content/en/docs/agent-prompt.md`, so
any host project depending on Modern Admin can extract it with
`bunx npm-skills extract` and have its AI coding agent (Claude Code,
Cursor, Aider, …) auto-load the framework's integration rules on
relevant tasks.
