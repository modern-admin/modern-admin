# @modern-admin/adapter-prisma

[![npm version](https://img.shields.io/npm/v/@modern-admin/adapter-prisma)](https://www.npmjs.com/package/@modern-admin/adapter-prisma)
[![license](https://img.shields.io/npm/l/@modern-admin/adapter-prisma)](https://github.com/modern-admin/modern-admin/blob/main/LICENSE)

> Prisma 7 adapter for Modern Admin — implements BaseDatabase/BaseResource on top of PrismaClient.

Part of [**Modern Admin**](https://github.com/modern-admin/modern-admin) — a universal, modern admin panel framework
built on NestJS + React 19, with Prisma/Drizzle adapters, Tailwind 4 UI, and
end-to-end Zod validation.

## Installation

```sh
bun add @modern-admin/adapter-prisma
```
