# @modern-admin/i18n

[![npm version](https://img.shields.io/npm/v/@modern-admin/i18n)](https://www.npmjs.com/package/@modern-admin/i18n)
[![license](https://img.shields.io/npm/l/@modern-admin/i18n)](https://github.com/modern-admin/modern-admin/blob/main/LICENSE)

> Translation registry and locale dictionaries for Modern Admin (9 locales).

Part of [**Modern Admin**](https://github.com/modern-admin/modern-admin) — a universal, modern admin panel framework
built on NestJS + React 19, with Prisma/Drizzle adapters, Tailwind 4 UI, and
end-to-end Zod validation.

## Installation

```sh
bun add @modern-admin/i18n
```

## Documentation

Setup guides, architecture, and usage examples live in the
