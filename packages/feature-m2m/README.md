# @modern-admin/feature-m2m

[![npm version](https://img.shields.io/npm/v/@modern-admin/feature-m2m)](https://www.npmjs.com/package/@modern-admin/feature-m2m)
[![license](https://img.shields.io/npm/l/@modern-admin/feature-m2m)](https://github.com/modern-admin/modern-admin/blob/main/LICENSE)

> Many-to-many relation property type for Modern Admin.

Part of [**Modern Admin**](https://github.com/modern-admin/modern-admin) — a universal, modern admin panel framework
built on NestJS + React 19, with Prisma/Drizzle adapters, Tailwind 4 UI, and
end-to-end Zod validation.

## Installation

```sh
bun add @modern-admin/feature-m2m
```

Setup guides, architecture, and usage examples live in the
