# @modern-admin/react

[![npm version](https://img.shields.io/npm/v/@modern-admin/react)](https://www.npmjs.com/package/@modern-admin/react)
[![license](https://img.shields.io/npm/l/@modern-admin/react)](https://github.com/modern-admin/modern-admin/blob/main/LICENSE)

> React 19 frontend layer for Modern Admin — translation boundary, routing, hooks.

Part of [**Modern Admin**](https://github.com/modern-admin/modern-admin) — a universal, modern admin panel framework
built on NestJS + React 19, with Prisma/Drizzle adapters, Tailwind 4 UI, and
end-to-end Zod validation.

## Installation

```sh
bun add @modern-admin/react
```

## Documentation

Setup guides, architecture, and usage examples live in the
