# @modern-admin/license

[![npm version](https://img.shields.io/npm/v/@modern-admin/license)](https://www.npmjs.com/package/@modern-admin/license)
[![license](https://img.shields.io/npm/l/@modern-admin/license)](https://github.com/modern-admin/modern-admin/blob/main/LICENSE)

> Ed25519/JWS license-key verification helpers for Modern Admin commercial packages.

Part of [**Modern Admin**](https://github.com/modern-admin/modern-admin) — a universal, modern admin panel framework
built on NestJS + React 19, with Prisma/Drizzle adapters, Tailwind 4 UI, and
end-to-end Zod validation.

## Installation

```sh
bun add @modern-admin/license
```

Setup guides, architecture, and usage examples live in the
